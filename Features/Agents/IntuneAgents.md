**Change Review Agent**  
Uses Microsoft Security Copilot's generative AI to evaluate Multi Admin Approval requests for PowerShell scripts on Windows devices. It provides risk-based recommendations and contextual insights to help administrators understand script behaviour and associated risks. I'll be concentrating on this agent for this blog post.  

https://gerryhampsoncm.blogspot.com/2025/12/my-first-look-at-intune-agents.html
https://learn.microsoft.com/en-us/intune/agents/change-review-agent

**Device Offboarding Agent**  
Identifies stale or misaligned devices across Intune and Entra ID, providing actionable insights and offboards devices subject to admin approval.  

https://gerryhampsoncm.blogspot.com/2025/12/my-first-look-at-intune-agents-continued.html  
https://learn.microsoft.com/en-us/intune/agents/device-offboarding-agent

**Policy Configuration Agent**  
Helps IT admins to translate complex requirements and industry standard documents into actionable Intune settings, and allows administrators to quickly generate Intune settings catalog policies.
